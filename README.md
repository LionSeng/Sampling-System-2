# 分层多阶段抽样系统

基于第七次全国人口普查（2020年）数据的分层多阶段抽样网页端 Demo。

## 四阶段抽样规格

| 阶段 | 内容 |
|------|------|
| 第一阶段 | 按地理+饮食习惯划分7大区，每区PPS随机抽2-3个省份 |
| 第二阶段 | 每省PPS随机抽：1个省会/特大城市 + 1个地级市 + 2个农村县 |
| 第三阶段 | 每城市/县内随机抽4个街道或乡镇 |
| 第四阶段 | 每街道/乡镇按年龄×性别分层，随机抽250人（依据2020年人口普查年龄性别分布） |

## 数据来源

- **省份人口**：第七次全国人口普查公报（国家统计局）
- **城市人口**：各省、市、区人口统计数据
- **年龄性别分层**：七普公报第五号（年龄结构）

## 技术栈

- 纯前端实现，无需服务器
- 支持 CSV/JSON 导出
- localStorage 保存历史记录

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。
